# RPG Character Management API

A GraphQL API for managing characters in a role-playing game, built with Apollo Server and TypeScript.


## Installation

1. Navigate to the backend directory:

   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application

### Development Mode

```bash
npm run dev
```

### Production Mode

```bash
npm run build
npm start
```

The server will start on `http://localhost:4000` with GraphQL Playground available at `http://localhost:4000/graphql`.

## Running Tests

```bash
# Run all tests
npm test

# Run tests in watch mode
npm run test:watch
```
